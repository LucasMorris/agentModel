Agent-Based Model

This Agent-Based Model has been created to help show how transportation preferences change between a fossil fuel car, electric car and bike, based upon environmental changes. Three default scenarios have been provided but custom values can be set.

The y-axis represents the percentage of the population that has chosen a specific mode of transportation. The x-axis shows time progression. The model has been set to run for 1000 ticks (unit of time).


How to use:

  • Select one of the three pre-made scenarios below or enter custom values.
  • If custom values have been entered into the input fields, enter key or the onscreen return icon must be pressed to confirm selection. This is not required when using sliders to choose values or when using the default scenarios.
  • Click “Start” to start the model and observe how the agent's preference changes overtime.
  • To reset/restart/change the environmental factors reload the page. If model has run and environmental factors have been manually changed,they will not influence the model. A page reload must be done to reset everything.
